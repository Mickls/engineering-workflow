# Out-of-scope 模板

记录已明确不要做、反复被误提、或历史上已证明不适合本项目的方案。

```md
# 非目标范围

## OOS-001 标题

- 状态：active / superseded
- 背景：
- 不采用的方案：
- 不采用原因：
- 替代方案：
- 再次考虑条件：
- 记录日期：
```

规则：

- 除非用户明确要求其他语言，非目标范围文档正文、标题、表格列名和状态解释都必须使用中文；文件名、命令、API、代码标识符、状态枚举和引用路径可以保留英文。
- 不要把所有暂时未做的事都写进 out-of-scope。
- 只有未来 agent 很可能重复提出、且会浪费时间或造成风险时才记录。
- 如果条件变化，更新“再次考虑条件”，不要把记录当成永久禁令。
